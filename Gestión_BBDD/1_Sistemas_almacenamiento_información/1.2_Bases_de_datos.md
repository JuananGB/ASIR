# <span style="color:lightcoral"> 1.2 Bases de Datos </span>

### Una <span style="color:lightcoral"> Base de Datos </span> es una colección de información perteneciente a un mismo contexto(o problema) , que está almacenada de forma organizada en ficheros.
### Una BBDD está organizada mediante tablas, que almacenan información concerniente a algún objeto o suceso. Estas tablas se relacionan formando vínculos o relaciones entre ellas, que ayudan a mantener la información de los diversos objetos de forma ordenada y coherente(sin contradicciones).

## <span style="color:lightcoral"> 1.2.1 Conceptos </span>

* ### <span style="color:lightcoral"> <b> Dato </b> </span> : Información concreta sobre algún concepto o suceso . Los datos se caracterizan por pertenecer a un tipo.
* ### <span style="color:lightcoral"> <b> Tipo de Dato </b> </span> : Indica la naturaleza del campo (numéricos , alfanuméricos , fecha)
* ### <span style="color:lightcoral"> <b> Campo <b> </span> : Identificador para toda una familia de datos. Cada campo pertenece a un tipo de datos. Al campo también se le llama columna.
* ### <span style="color:lightcoral"> <b> Registro </b> </span> : Es una recolección de datos referentes a un mismo concepto o suceso. A los registros también se les llama tuplas o filas .

* ### <span style="color:lightcoral"> <b> Campo Clave </b> </span> : Es un campo especial que identifica de forma única a cada registro. Así, el NIF es único para cada persona, por tanto es campo clave.

* ### <span style="color:lightcoral"> <b> Tabla </b> </span> : Es un conjunto de registros bajo un mismo nombre que representa el conjunto de todos ellos .

* ### <span style="color:lightcoral"> <b> Consulta </b> </span> : Es una instrucción para poder hacer peticiones a una base de datos.

* ### <span style="color:lightcoral"> <b> Índice </b> </span> :  Es una estructura que almacena los campos clave de una tabla, organizándolos para hacer más fácil encontrar y ordenar los registros de esa tabla.

* ### <span style="color:lightcoral"> <b> Vista </b> </span> : Es una transformación  que se hace a una o más tablas para obtener una nueva tabla. Esta nueva tabla es una tabla virtual.

* ### <span style="color:lightcoral"> <b> Informe </b> </span> : Es un listado ordenado de los campos y registros seleccionados en un formato fácil de leer.

* ### <span style="color:lightcoral"> <b> Guiones o scripts </b> </span> : Son un conjunto de instrucciones, que ejecutadas de forma ordenada, realizan operaciones avanzadas de mantenimiento de los datos almacenados en la base de datos.

* ### <span style="color:lightcoral"> <b> Procedimientos </b> </span> : Son un tipo especial de script que está almacenado en la base de datos y que forma parte de su esquema .

## <span style="color:lightcoral"> 1.2.2 Estructura de una BBDD </span>

### Una base de datos almacena los datos a través de un esquema. Dicho esquema contiene (campos,registros,procedimientos,vistas,etc...) . Al esquema también se le suele llamar <span style="color:lightcoral"> metainformación </span>
### BBDD como MySQL y DB2 almacenan el esquema de la bbdd en tablas .
