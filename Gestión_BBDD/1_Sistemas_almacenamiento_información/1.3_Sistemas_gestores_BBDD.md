# <span style="color:lightcoral"> 1.3 Los Sistemas Gestores de Base de Datos </span>
## <span style="color:lightcoral"> 1.3.1 Concepto de Sistema Gestor de Base de Datos </span>

### Se define un Sistema Gestor de Base de Datos (SGBD), como el conjunto de herramientas que facilitan la consulta, uso y actualización de una base de datos.

## <span style="color:lightcoral"> 1.3.3 El lenguaje SQL </span>
### Lenguaje sencillo en el cuál el usuario realiza preguntas al servidor, contestando este a las demandas del usuario, este lenguaje se denomina SQL . (se divide en 4 sublenguajes).

* ### <span style="color:lightcoral"> **Lenguaje DML** </span> : Este lenguaje permite 4 sentencias sencillas("SELECT","INSERT","UPDATE","DELETE").

* ### <span style="color:lightcoral"> **Lenguaje DDL** </span> : Este lenguaje permite crear toda la estructura de una base de datos ("DROP","CREATE").

* ### <span style="color:lightcoral"> **Lenguaje DCL** </span> : Incluye ("GRANT" y "REVOKE") que permiten al administrador gestionar el acceso a los datos contenidos en la base de datos.

* ### <span style="color:lightcoral"> **Lenguaje TCL** </span> : Permitit ejecutar varios comandos de forma simultánea como si fuera un comnando atómico o indivisible ("COMMIT" , "ROLLBACK").
