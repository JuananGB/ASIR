# Comandos Python 

## Crear entorno virtual:

Desde la raíz del proyecto creamos la carpeta con todo el entorno virtual dentro 

```bash
python -m venv venv
```

La activamos : 

```bash
source venv/bin/activate
``` 

Para desactivarla :

```bash
deactivate
```

Herramientas interesantes dentro de /venv :

```bash
pip install pylint black jupyter  
```

* pylint, analiza tu código Python para encontrar errores,

* black, formatea automáticamente tu código Python con un estilo limpio, consistente y estándar.

* jupyter, permite escribir y ejecutar código Python de forma interactiva, en celdas, junto con texto, imágenes y gráficos.
