| Lenguaje                                           | Significa / Origen              | Uso principal                                                              |
| -------------------------------------------------- | ------------------------------- | -------------------------------------------------------------------------- |
| **HTML** (*HyperText Markup Language*)             | El lenguaje de la Web           | Estructura páginas web (junto con CSS y JavaScript).                       |
| **XML** (*eXtensible Markup Language*)             | Lenguaje extensible para datos  | Intercambio de información entre sistemas, configuración, documentos, etc. |
| **XHTML** (*eXtensible HTML*)                      | Versión XML de HTML             | Combina la estructura de HTML con la sintaxis rigurosa de XML.             |
| **SVG** (*Scalable Vector Graphics*)               | Basado en XML                   | Representa gráficos e imágenes vectoriales en la web.                      |
| **MathML** (*Mathematical Markup Language*)        | Basado en XML                   | Representa fórmulas y notación matemática.                                 |
| **KML** (*Keyhole Markup Language*)                | Basado en XML                   | Define información geográfica (usado por Google Earth y mapas).            |
| **RSS / Atom**                                     | Basados en XML                  | Distribución de contenidos (feeds de noticias, blogs).                     |
| **Markdown (MD)**                                  | Lenguaje de marcas ligero       | Usado para documentación, README en GitHub, y texto web fácil de leer.     |
| **LaTeX**                                          | Lenguaje de marcas tipográfico  | Creación de documentos científicos y académicos con formato avanzado.      |
| **JSON (no es de marcas, pero similar propósito)** | Formato de intercambio de datos | Alternativa moderna a XML por su simplicidad.                              |
